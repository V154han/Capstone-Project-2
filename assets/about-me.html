Who Am I?

Hello and welcome to my profile! Now I have started my newest journey in the world of coding - and that is web development! This is my first ever website that I've built purely from HTML and CSS and have tried to minimize the amount of bootstrap that I'm using to really familiarze myself with using CSS. I'm looking forward to have you join me in my journey!
